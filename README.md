# m4bon

**Beat-Oriented Note Entry for MuseScore 4**

A plugin that lets you define rhythmic patterns (beat divisions, durations) first, then assign pitches — a "rhythm-first" approach to note input.

## Quick Start

```bash
ln -s "$PWD" ~/Documents/MuseScore4/Plugins/m4bon
```

Then enable the plugin in **Home → Plugins → m4bon** and run it via **Plugins → m4bon**.
